# 📦 Amazon Sales Analytics & Machine Learning Suite

An interactive Streamlit application built using Amazon Sales data for Data Analysis, Visualization, and Machine Learning.

## Features

### 📊 Dashboard
- Total Orders
- Total Revenue
- Category-wise Sales Analysis

### 💰 Amount Prediction
- Predict sales amount using Random Forest Regression

### 🚚 Order Status Prediction
- Predict order status using Machine Learning

### ❌ Cancellation Prediction
- Predict cancellation risk based on order details

### 👥 Customer Segmentation
- Segment customers using K-Means Clustering

### 📈 Sales Forecasting
- Forecast future sales trends using historical sales data

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Plotly
